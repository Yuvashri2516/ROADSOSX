# RoadSoS AI & Computer Vision Module

Features: YOLOv8 object detection, collision risk scoring, lane monitoring, safe drowsiness detection, Flask UI.

## Run
```powershell
pip install -r requirements.txt
python app.py
```
Open http://127.0.0.1:5000

## Note
For demo speed, the app skips frames and processes max 220 frames. Use short MP4 videos for best results.
